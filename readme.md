---


You need to run `npm run dev` to fire the server. It's much fast than CRA


### GIT COMMANDS
`git pull` - do this initially and whenever you have merged and return to main branch to makes sure you have all changes
<br>

`git checkout -b '<branch-name>'` - this to create and go to a branch
<br>

`git add .` - save changes
<br>

`git commit -m 'your message here'` - commit changes to working tree with a message
<br>

`git push -u origin '<branch-name>'` - push up your work. When finished you can create a pull request
<br>
